## What is Screen Reader Mode?

Screen Reader Mode transforms your web browsing into a fully voice-guided experience. It announces browser actions, reads webpage elements, and allows complete keyboard-only navigation - perfect for accessibility or hands-free browsing.

## ğŸ“ Full Shortcut Reference

### Navigation (Move Around Page)
- **J** - Next element
- **K** - Previous element
- **N** - Next heading (H1, H2, etc.)
- **P** - Previous heading
- **L** - Next link
- **B** - Next button
- **I** - Next image
- **F** - Next form field (input, textarea, select)

### Information (Read Page Details)
- **T** - Read page title
- **U** - Read current URL
- **R** - Repeat current element
- **H** - Show help menu (lists all shortcuts)

### Control (Manage Speech)
- **S** - Stop speaking
- **Escape** - Stop speaking (alternative)
- **Ctrl+Shift+S** - Toggle Screen Reader on/off

### Browser Actions (Announced Automatically)
- **Ctrl+T** - New tab
- **Ctrl+W** - Close tab
- **Ctrl+R** - Refresh page
- **Ctrl+F** - Find dialog
- **Ctrl+P** - Print dialog
- **Ctrl+D** - Bookmark page
- **Alt+Left** - Go back
- **Alt+Right** - Go forward
- **Ctrl++** - Zoom in
- **Ctrl+-** - Zoom out
- **Ctrl+0** - Reset zoom
